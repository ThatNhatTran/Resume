# assets/

Drop your media here:

- **`photo.jpg`** — a profile photo. To show it, open `index.html`, find the
  `hero__avatar` block, and replace it with:
  ```html
  <img class="hero__photo" src="assets/photo.jpg" alt="Your Name" />
  ```

- **`resume.pdf`** — a downloadable résumé. To use a real PDF instead of the
  built-in "print to PDF" button, replace the `#downloadBtn` button in
  `index.html` with:
  ```html
  <a class="btn btn--primary" href="assets/resume.pdf" download>Download Résumé</a>
  ```
